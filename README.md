# Fullstack Developer Capstone

## 📌 Project Overview

**Fullstack Developer Capstone** is a complete full-stack web application demonstrating modern development practices across the frontend, backend, database, and deployment pipeline.

The goal of this project is to showcase the ability to design, build, and deploy a scalable full-stack application.

---

## 🚀 Features

* User authentication and authorization
* RESTful API integration
* CRUD operations
* Responsive frontend UI
* Database integration
* Error handling and validation
* Deployment-ready configuration

---

## ⚙️ Installation

### server

```bash
pip install virtualenv
virtualenv djangoenv
source djangoenv/bin/activate

python3 manage.py runserver

localhost:8000
```

### database 
```bash
docker build . -t nodeapp
docker-compose up  

localhost:3030
```

### frontend
```bash
npm install
npm run build
```
